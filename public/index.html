统一输入中心

手动 / 表格 / 数据库 / API / 语音按钮
手动输入

适合锚定目标、知识库、私信、评论处理和成交话术的即时录入。

表格导入

适合批量导入客户画像、评论样本、知识条目、线索清单和话术素材。

数据库导入

预留 CRM、商城、医院 HIS、门店系统和自有私域数据库接入。

API 接口导入

预留各平台 Webhook、开放平台接口和内部系统接口。

AI 语音输入

不再单独占一个大模块；所有文字输入框旁都有语音按钮，点击即可把语音转成文字。

数据导入/导出中心

导入 / 导出 / 模板 / 数据库

知识库导入

用模板批量整理行业知识、FAQ、案例、资质和成交话术。

评论私信样本导入

导入抖音、小红书、快手、视频号、电商客服的留言和私信样本，用于分级和生成话术。

目标画像导入

导入品牌、区域、竞品、人群、消费行为、媒体偏好和成交目标。

数据库接入

预留 CRM、企微客户池、商城订单、表单系统、医院/门店业务系统的数据入口。

数据导出

导出知识库、导入模板和复核材料，方便备份、迁移和人工校对。

人工呼叫导入云客

通话记录 / 意向分级 / 跟进人

导入字段

客户称呼、手机号、来源平台、通话时间、通话结果、意向等级、跟进人、下次跟进时间和备注。

导入前校验

默认先去重、校验手机号格式、检查客户授权来源，并对敏感字段做脱敏预览。

云客承接规则

高意向进入优先跟进池，中意向进入培育池，负面或投诉类进入人工复核队列。

导入到CRM系统

字段映射 / 去重 / 同步策略

CRM字段映射

线索编号、客户名称、手机号、平台来源、客户阶段、标签、跟进人、最近互动和成交状态。

同步策略

支持新增、更新、合并重复客户、失败重试和同步审计；真实写入前必须配置 CRM API 凭证。

数据边界

不在前端显示密钥明文；含手机号、微信号和病患信息等敏感数据必须先确认授权来源。

外设接口

手机 / 语音 / 平台终端

企业微信与个人微信终端

预留企业微信、个人微信、EasyClaw、群机器人和私域承接终端。

内容平台客服终端

预留抖音、快手、小红书、视频号、B站、知乎、头条号和百家号。

电商客服终端

预留淘宝、拼多多、京东等平台内客服和售后消息。

语音与录音终端

预留麦克风、通话录音、会议录音和客服录音转写，默认需要明确授权。

API接口中心

凭证 / Webhook / 回调

平台开放 API

集中管理企业微信、抖音、快手、小红书、视频号、电商和海外平台的授权状态。

业务系统 API

预留 CRM、订单、商城、预约、会员和复购系统的接口。

Webhook 回调

预留平台消息推送、表单提交、支付结果、用户行为和客服消息回调。
